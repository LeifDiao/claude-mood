为什么零 token？Why zero token?

它只是在你电脑里翻了翻 Claude Code 自己写的日记，然后画了张面板。It just reads the diary Claude Code already keeps on your disk, and draws a panel.
  • 只读本地文件Reads local files only — 服务端只读取 ~/.claude/projects/**/*.jsonl。The server only reads ~/.claude/projects/**/*.jsonl.
  • 从不调用模型 / APINever calls a model/API — 不连 Anthropic、不发任何请求，不花一分钱 token。No requests to Anthropic or any model. Not a single token.
  • 只绑 localhostLocalhost only — 前端只向自己的源拿数据，无 CDN、外部字体或图片。The page only fetches from its own origin; no CDNs, fonts, or images.
